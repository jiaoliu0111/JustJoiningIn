# 凑热闹微信小程序首版设计

## 背景

目标产品是一个面向周末防宅家、低压力放松人群的微信小程序。核心不是旅游攻略、相亲交友或精致打卡，而是帮用户在周末用最低决策成本走出家门，找到附近免费、轻松、有人气但不强社交的活动、路线和临时搭子。

首版选择原生微信小程序实现，使用本地 mock 数据跑通主要体验。后续可将 mock 数据替换为真实定位、地图、官方活动和组队接口。

## 产品原则

- 一打开就知道附近有什么热闹，不要求用户先规划。
- 所有行动都要低压力：无需准备、就近、短时效、可随时退出。
- 结伴是纯陪伴，不做恋爱、相亲、粉丝关注、复杂社交资料。
- 推荐内容以免费或低消费、易到达、上海本地公园/社区/街区为主。
- 氛围要接地气、松弛、真实，避免网红化、精致化、商业化。

## 首版范围

首版采用“首页闭环 MVP”。它要表达完整第一感受：附近有低成本热闹，用户不用规划，也不用社交用力，点一下就能出门。

包含功能：

- 即刻凑局：三种极简出行状态，支持静音同行和 1 小时短时效。
- 附近热闹：地图感首页区域和热闹点列表，展示附近公园、步道、街区、市集。
- 上海官方活动：公园活动和社区活动 mock 聚合，展示来源、时间、地点和报名提示。
- 现成放空路线：家周边 3km 内、免费、不累脚的路线卡。
- 临时轻组队：最多 3 人、短时匿名、一键散局。
- 我的偏好：最大距离、默认匿名、是否接受静音同行、极简出门清单。

暂不包含：

- 真实登录、真实聊天、支付、实名认证。
- 真实地图 API、真实导航、真实活动接口或爬取。
- 旅游攻略、长途出行、相亲交友、粉丝关注、探店带货、付费活动。
- 复杂资料填写、头像审核、排行榜、积分、点赞评论。

## 信息架构

小程序包含 4 个 tab 和 1 个复用详情页。

### 首页

首页是核心引流页，按“立刻出门”的心理路径组织：

1. 周末低压力问候和宅家轻提醒。
2. 即刻凑局主按钮。
3. 附近热闹地图感卡片。
4. 当前可加入的临时凑局。
5. 上海官方活动精选。
6. 现成放空路线。
7. 极简出门清单。

### 热闹

热闹页承接“不知道去哪”的需求，提供列表化浏览和轻筛选。

筛选标签：

- 免费
- 3km 内
- 周末
- 静音友好
- 公园活动
- 社区活动

内容包括官方活动、附近热闹点和推荐路线。点击卡片进入详情页。

### 凑局

凑局页承接“想出门但没人陪”的需求。

主要内容：

- 三种发起状态：没事出门晃一晃、就近散步压马路、随缘搭伴随便逛。
- 可切换静音同行。
- 时长默认 1 小时，可选半天。
- 当前临时队伍列表，最多 3 人。
- 匹配成功态，显示低压力行动提示和一键散局。

### 我的

我的页不做复杂个人资料，只保留减负和安全设置。

内容包括：

- 最大匹配距离，默认 3km。
- 默认匿名模式。
- 是否接受静音同行。
- 是否开启周末轻提醒。
- 极简出门清单：舒服的鞋、水杯、轻装出行。

### 详情页

详情页复用展示四类对象：

- 官方活动
- 热闹地点
- 放空路线
- 临时凑局

详情页根据类型显示时间、距离、地点、来源、适合原因、参与状态和主行动按钮。官方活动和地点可一键带入“去这里凑热闹”的发起流程。

## 核心交互

### 即刻凑局

用户从首页或凑局页进入发起面板，只需要做少量选择：

- 选择出行状态。
- 选择是否静音同行。
- 确认时长，默认 1 小时。
- 点击发布。

发布后进入匹配成功态，显示模拟匹配结果、队伍人数、预计距离和一键散局按钮。

### 附近热闹

首页用 WXSS 绘制地图感区域，包括道路、绿地、河岸和点位。点位不是实时地图，只是帮助用户建立“附近有去处”的感知。点击点位或列表卡片进入详情。

### 官方活动

官方活动以 mock 数据展示。每条活动包含来源字段，例如“上海公园官方发布”“社区云活动”。详情页显示来源，避免用户感觉信息不可靠。

### 现成放空路线

路线以轻任务方式呈现。用户点击路线后看到路线长度、预计时长、轻松程度、途经点和开始出行按钮。首版不做真实导航，点击开始后显示“出门清单”和进行中状态。

### 一键散局

组队进行中时，用户可以点击一键散局。首版本地切换为已散局状态，并显示系统代通知文案，避免用户需要解释。

## 视觉方向

参考图传达的是暖阳、街区、公园、手机地图和轻松出门。小程序会采用更适合真实使用的轻量表达：

- 主色：暖橙、奶白、草绿色。
- 辅色：浅黄、河岸蓝、柔和灰棕。
- 卡片圆角控制在 8px 左右，底部 tab 和主要按钮保持清晰。
- 首页顶部使用暖光渐变，但不做沉重插画背景。
- 地图感区域用 WXSS 绘制，减少图片依赖。
- 文案口吻松弛、接地气，避免营销化和鸡血化。

关键标签：

- 免费
- 3km 内
- 无需准备
- 静音同行
- 1 小时散局
- 官方来源

## 验证

实现后进行以下验证：

- 检查小程序必要文件完整。
- 校验 JSON 文件语法。
- 校验 JS 文件能被 Node 解析。
- 检查 WXML 标签闭合和页面引用路径。
- 用微信开发者工具打开项目进行最终渲染确认。

## 成功标准

- 用户打开首页能在 5 秒内理解产品是“周末就近凑热闹、低压力出门”。
- 首页能从即刻凑局、附近热闹、官方活动、现成路线四个入口进入下一步。
- 所有组队相关界面都明确表达短时、匿名、静音、可散局。
- 视觉气质接近参考图的暖阳和街区感，但界面保持清晰可用。
- 代码结构清楚，后续接真实接口时不需要重写页面。
