# 凑热闹小程序新需求改版设计

## 依据

本轮以 `/Users/song/Desktop/hotspot-app-DESIGN.md` 为准。产品目标从“临时结伴/组队”调整为“10 秒找到附近、免费、不尴尬、马上能出门的小理由”。

## 改版原则

- 强调附近、免费、马上能去、低压力。
- 弱化陌生人社交，不出现匹配搭子、发布凑局、成员数、一键散局、匿名结伴等表达。
- 页面像本地生活发现工具，不像派对、约会或严肃活动平台。
- 地图 UI 用浅奶油底、沙米色路网、草绿绿地、地图橙活动气泡。
- 活动卡片按“活动名 → 距离/时间 → 免费标签 → 尴尬度降低说明”组织。

## 页面调整

首页改为“附近小热闹”首屏：搜索框文案为“附近有什么免费小热闹？”，地图气泡显示具体小活动，CTA 为“给我一个出门理由”。

热闹页改为“附近正在发生的小事”：筛选为免费、步行 10 分钟内、现在开始、不用报名、路过看看、安静友好。

原“凑局”tab 改名为“理由”：页面改成随机推荐出门理由，提供“换一个近点的”和“现在去看看”，不再发布组队或匹配。

详情页统一展示活动、距离、时间、免费/不用报名/路过看看等信息，主按钮为“现在去看看”。

我的页改为轻偏好：只看免费、步行范围、不用报名优先、安静路过友好，保留极简出门清单。

## 数据调整

新增 `nearbyMoments` 作为核心活动数据，字段包含活动名、气泡文案、步行分钟、时间状态、是否免费、是否不用报名、低压力说明、标签和详情描述。

新增 `reasonCards` 作为随机出门理由数据。

保留 `routes` 作为低压力路线补充。
